# PetShop — Plataforma Online de Petshop

Este projeto foi desenvolvido como parte da disciplina de sistemas web e tem como objetivo criar uma plataforma web para um petshop.

##Produtos

Os produtos expostos são:
- **Acessórios**: Guia Laranja PetAmor, Coletinho Fashion Pet
- **Rações**: Ração Golden Special, Ração True Nature
- **Higiene e Limpeza**: Shampoo Filhote Suave, Shampoo Aumigos

##Serviços

A página de serviços permite agendar:
- Banho e tosa
- Banho e tosa com tele-busca
